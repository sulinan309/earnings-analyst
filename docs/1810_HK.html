Executive Summary

建议动作: 回避/减持 — 当前32 HKD高于乐观估值(27 HKD)约19%，在v4.3框架下明确超出估值上限
2025年报整体符合'史上最强'叙事——全年营收4573亿(+25%)，经调整净利润392亿(+44%)，汽车业务首破千亿且年度经营收益转正至9亿。然而Q4单季暴露的结构性隐忧不容忽视：手机毛利率跌至8.3%历史低位，Q4经营现金流仅6亿。 v4.3估值修正：Guard Rail要求OE≤1.2×FCF。FCF估计~200亿(远低于经调整净利润392亿)——OE上限240亿。分层估值：Layer A(手机+IoT+互联网) OE~200亿，Layer B(汽车+AI) OE~0-60亿。基准OE 240亿(g=3%, r=11%) → 基准内在价值22 HKD。当前32 HKD高出45%。 Payback 30年(base, static)是三家中最长——每投入1港币需要30年通过现金流回收。按李录/价值框架，32 HKD处于回避区。400股@33.47应定位为swing trade而非价值持仓。34.5+考虑减持，释放资金优先配置快手(明确加仓区)和腾讯(等回落)。
TL;DR:
  • v4.3估值：保守16 / 基准22 / 乐观27 HKD。当前32处于回避区(>27)，Payback 30年(三家最贵)
  • 核心问题：经调整净利润392亿 vs FCF仅~200亿——Guard Rail要求OE≤240亿，远低于市场隐含的利润预期
  • 分层估值：手机×AIoT×互联网(Layer A) OE~200亿 + 汽车×AI(Layer B) OE~0-60亿
  • 按李录/价值框架不应持有。400股@33.47定位为swing trade(非价值买入)，34.5+考虑减持释放资金给快手/腾讯

Key Forces（决定性力量）

#1 汽车业务从'烧钱'到'利润贡献者'的跨越速度
2025年已实现年度经营收益转正。2026年目标交付55万辆，北京+武汉双工厂总产能~120万辆。汽车业务24.3%的全年毛利率已高于手机×AIoT分部的21.7%，如果汽车规模持续放大，将结构性地拉升集团毛利率。新一代SU7开售34分钟锁单15000台。
OE含义: 汽车Capex目前大部分是扩张性的（不计入OE），但随着规模化，折旧替换部分将逐步转为维持性Capex，需持续跟踪。
#2 手机业务的成本-收入剪刀差（存储涨价危机）
存储芯片长周期涨价（管理层判断可能持续到2027年底）正在侵蚀手机毛利。2025年手机毛利率从12.6%降至10.9%，Q4更跌至8.3%。手机仍贡献41%的收入，如果毛利率持续承压，将显著拖累集团利润增速。
OE含义: 手机毛利率每下降1pct，约减少19亿利润。如果毛利率长期维持8-9%，OE将被压缩~60-80亿。
#3 AI生态 + '人车家'协同效应的货币化
MiMo大模型全球排名第八、国内第二。miclaw AI Agent正在封测。AI眼镜出货全球第三。但这些目前均不构成独立收入线——AI是费用项不是收入项。未来三年600亿AI投入意味着：如果AI未能有效货币化，将持续压制利润增速。
OE含义: 600亿AI投入/3年 = 年均200亿费用项。如果归类为扩张性Capex不影响OE，但如果被视为维持竞争力的'必要投入'则应计入维持性Capex。

Variant View — 市场共识 vs 我们的判断

市场共识
市场共识：汽车业务是最大看点，2026年55万辆目标令人兴奋，但9000亿市值(~23x经调整PE)已部分price in汽车故事。
市值隐含利润需年化增长~20%+
我们的判断
OE框架下内在价值仅6804亿 vs 市值9000亿，安全边际-24.4%。核心问题：汽车Capex(二期工厂+AI)正在吞噬OE。2025年汽车经营利润仅9亿，但消耗的Capex远超此数。市场在为'汽车年销100万辆'定价，但从OE视角这个故事需要更多季度的现金流验证。
市场错在哪
市场被汽车叙事绑架了。用PE看小米（23x）和用OE看小米（-24.4%）给出完全相反的结论。分歧在于：如果汽车Capex是高回报扩张性投资，PE才有意义；如果回报不确定，应该用更保守的OE框架。Q4经营现金流仅6亿是一个警告信号。
关键分歧
  • 汽车Capex是'高回报投资'还是'军备竞赛'？单车利润拐点已到但规模待验证
  • Q4经营现金流骤降至6亿是季节性还是结构性？
  • 配股425亿港币后ROE被稀释，增长能否消化稀释效应？
正面催化剂
